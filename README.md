# AWS Cost Estimator (EC2 Hours)

A simple serverless web application that estimates AWS EC2 costs in real time.

---

## 🌐 Live Demo
http://matthew-aws-cost-estimator-2026-april.s3-website-us-east-1.amazonaws.com/

---

## 📌 What This Project Does

Users enter estimated EC2 usage hours and receive a real-time cost calculation powered by an AWS backend.

This simulates how cloud pricing tools work in production environments.

---

## 🧱 Architecture

Frontend (HTML/JS)
→ Amazon S3 (Static Hosting)
→ API Gateway (HTTP endpoint)
→ AWS Lambda (Python logic)
→ Response returned to UI

---

## ⚙️ Tech Stack

- AWS Lambda
- API Gateway
- S3 Static Hosting
- HTML / JavaScript
- Python (backend logic)

---

## 💡 Key Learning Outcomes
