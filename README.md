# AWS Cost Estimator (EC2 Hours)

A simple web-based AWS cost estimator built using:

- AWS Lambda (backend logic)
- API Gateway (HTTP endpoint)
- S3 Static Website Hosting (frontend)
- HTML + JavaScript (UI)

## 🌐 Live Demo
http://matthew-aws-cost-estimator-2026-april.s3-website-us-east-1.amazonaws.com/

## 🚀 Features
- Enter EC2 usage hours
- Get real-time cost estimate
- Fully serverless architecture
- Hosted on AWS

## 🧠 Architecture
Frontend → S3 → API Gateway → Lambda → Response

## 📌 Purpose
This project demonstrates building and deploying a full serverless web application using AWS services.
